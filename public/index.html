How does it work?

So Make It is a small charity. Members pay a small monthly fee to So Make It, and then they can use the So Make It Makerspace which has space, storage, and various tools that are available.

Community. Not for profit, friendly, and fun.

We believe that everyone, from any background, should be able to learn new skills and make new friends. To enable this, we run a Makerspace where members can use our resources and tools to create amazing new things. Our members love to share knowledge and skills, so lack of experience is not a problem!
